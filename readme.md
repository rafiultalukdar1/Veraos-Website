# Veraos – Corporate Website

Veraos is a modern, fully responsive corporate website converted from Figma into clean, production-ready code. It presents a high-end AI-driven platform interface with a strong focus on performance, user experience, and conversion-driven design.

## Overview

Built with HTML, CSS, JavaScript, and jQuery, the website delivers smooth interactivity across all devices. The codebase is pixel-perfect, well-structured, and optimized for cross-browser compatibility.

Advanced libraries like Particles.js and WOW.js add engaging animations and visual depth, while modal integration improves overall user interaction.

## Tech Stack

- HTML
- CSS
- JavaScript
- jQuery
- Particles.js
- WOW.js

## Features

- Fully responsive design
- Pixel-perfect Figma to HTML conversion
- Clean and maintainable code structure
- Cross-browser compatibility
- Smooth animations and interactive UI
